## Overview
パワーポイントのマクロを使って全てのページのフォントを統一する

## Requirement

- Office 2016(PowerPoint)
- VBA

## Usage

1. ファイルを起動する
1. スライドショーモードにする
1. ボタンを押して、テキスト、表、シェイプ文字、グループ結合で文字が変更されている事を確認する

## Features

ボタンの文字表示が汚い
